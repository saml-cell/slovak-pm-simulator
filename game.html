Ako získať Groq API kľúč (zadarmo):

  1. Choď na console.groq.com
  2. Zaregistruj sa (stačí Google účet)
  3. Klikni na API Keys v ľavom menu
  4. Klikni Create API Key
  5. Skopíruj kľúč a vlož ho sem hore

Groq je zadarmo (rate limit ~30 req/min). AI analyzuje tvoje politiky realistickejšie než lokálny systém.

Ako získať Anthropic API kľúč:

  1. Choď na console.anthropic.com
  2. Zaregistruj sa a pridaj platobnú metódu
  3. Choď do Settings → API Keys
  4. Vytvor nový kľúč a vlož ho sem hore

Anthropic je platený (~$0.001 za ťah). Najkvalitnejšia AI analýza.
